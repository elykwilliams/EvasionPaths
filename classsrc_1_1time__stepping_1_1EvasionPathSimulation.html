Detailed Description

This class provides the main interface for running a simulation.

It provides the ability to preform a single timestep manually, run until there are no possible intruders, or until a max time is reached. evasion_paths provides the name of the transitions per timestep, is_connected is a flag that will be true as long as the graph remains connects.
