Detailed Description

Provide abstract base of features that a boundary must satisfy.

The boundary class generates the positions of the boundary sensors, a way of specifying if a given point is in or out of the boundary. It will also know the boundary cycle associated with the outside of the fence (the "alpha_cycle". And finally, it will be the class to generate initial data since it can determine how best go generate random points inside the domain.
